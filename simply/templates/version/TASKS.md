# Tasks

Flat backlog across all phases. Phase is an attribute, not a container — group, sort, or filter by the Phase column.

| Status | Pri | Phase | ID | Task | Description |
|--------|-----|-------|----|------|-------------|
| ⬜ | P0 | 01 | T1 | [Task name](#t1-task-name) | {Short description} |

Status: ⬜ todo, 🔄 in-progress, ✅ done, 🚫 blocked, ⏸️ deferred

Phase identifier is freeform — `01`, `02-auth`, `polish`, whatever. If you want sort order, prefix with numbers. Task IDs (`T1`, `T2`, …) are global and stable — they survive moving a task between phases.
